MayaMystic API Framework

Enterprise-grade middleware-based networking SDK for Unity. Designed for scalable production systems using clean architecture and middleware execution pipelines.

Core Features

  • Async HttpClient-based ApiManager
  • Middleware Execution Pipeline
  • Authentication Injection
  • Smart Retry Middleware
  • Token Provider Abstraction
  • Endpoint Resolver System
  • Strongly Typed JSON Serialization

Architecture Flow

Application Layer
    └── ApiHandler

Infrastructure Layer
    ├── ApiManager
    ├── Middleware Pipeline
    │     ├── LoggingMiddleware
    │     ├── AuthMiddleware
    │     ├── SmartRetryMiddleware
    └── HttpClient
