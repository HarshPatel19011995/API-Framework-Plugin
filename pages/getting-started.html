Getting Started

This guide walks you through integrating MayaMystic API Framework into your Unity project.

Step 1 — Install Package

Add package via Unity Package Manager
or place inside Packages/ folder.

Step 2 — Create ApiManager

var apiManager = new ApiManager();

Step 3 — Register Middleware

apiManager.UseMiddleware(new LoggingMiddleware());
apiManager.UseMiddleware(new AuthMiddleware(tokenProvider));
apiManager.UseMiddleware(new SmartRetryMiddleware());
